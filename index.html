Regex Cheat Sheet

  • \d - digit
  • \w - word character
  • \s - whitespace
  • . - any character
  • * - 0 or more
  • + - 1 or more
  • ? - optional
  • ( ) - capture group
  • [abc] - character set
  • ^ - start of string
  • $ - end of string
